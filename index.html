Wild Knots (3 min)

It is easy to tie a necklace, but if it is made of spheres, and these spheres are mirrors, we can observe a phenomenon similar to what we see when we put two parallel mirrors face to face. On each sphere we can see the reflection of all the others, reproducing the knot on a smaller scale inside itself. Recursively, each reflected sphere is also a mirror that contains the reflection of all the other spheres, and we can continue this process indefinitely. At the end, what we see is this, a wild knot.

Animation: Aubin Arroyo

Music: Meret Becker / Gläsernes Gesicht

Nœuds sauvages (3 min)

Il est facile de nouer une cravate, mais si la cravate est composée de sphères, et si les sphères sont des miroirs, on observe un phénomène analogue à ce qu’on voit quand on place face à face deux miroirs parallèles. Sur chaque sphère, on voit la réflexion de toutes les autres, reproduisant le nœud à une plus petite échelle dans chacune. Ainsi de suite, chaque sphère réfléchie est aussi un miroir qui contient la réflexion de toutes les autres sphères, et on peut continuer ce processus indéfiniment. A la fin, ce qu’on voit est un nœud sauvage.

Animation: Aubin Arroyo

Musique: Meret Becker / Gläsernes Gesicht

Roli’s cube (4 min)

A walk through a chiral polytope of low dimension.
This object is a compound of twelve helices wrapped around the structure of a four-dimensional cube, also known as tesseract. Each helix is in contact with all the others but one, which shares the same colour. This arrangement of helices has a rich set of symmetries closely related with the rotations of the tesseract. Interestingly, it contains no mirror reflections, which is a unusual property on objects with such level of symmetries.

Author: Aubin Arroyo

Le cube de Roli (4 min)

Une promenade sur un polytope chiral de petite dimension. Cet objet est formé de douze hélices, enroulées autour d’un hypercube de dimension quatre, appelé aussi tesseract. Chaque hélice touche toutes les autres, sauf une qui a la même couleur. Cet arrangement d’hélices possède un riche ensemble de symétries qui ont un rapport avec les rotations du tesseract. Il est intéressant de noter que ce complexe d’hélices n’admet pas de symétrie miroir, ce qui est assez inhabituel pour un objet aussi symétrique.

Auteur: Aubin Arroyo
